# Serialization string

## Installation

```bash
npm i @ffsm/serialize
```

OR

```bash
yarn add @ffsm/serialize
```

## Usage

Using single function

```ts
import { encode } from '@ffsm/serialize`;

encode('&');
```

OR using with default object

```ts
import Serialize from '@ffsm/serialize';

Serialize.encode('&');
```
